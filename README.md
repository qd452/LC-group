# LC-group

## Quick Start

### Backend

```sh
python3 venv venv
source ./venv/bin/activate
pip install -r requirements.txt
python web_runner.py
```

To deactivate, just run `deactivate`

### Fronten

```sh
npm start
npm run build # to create a production build,
```
